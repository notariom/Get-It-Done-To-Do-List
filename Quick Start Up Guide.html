Get It Done! To-Do List: Quick Start Up Guide

This guide provides an overview of how to use the "Get It Done! To-Do List" application.

1. Application Overview

The "Get It Done! To-Do List" is a simple web-based application to help you manage your tasks. It offers:

  • Task Management: Easily add, edit, and delete your tasks.
  • Completion Tracking: Mark tasks as complete or incomplete with a click.
  • Persistence: Your tasks are automatically saved in your browser's local storage.
  • Clean Design: A streamlined visual design for easy readability.

2. Accessing the Application

To get started, simply:

  • Locate the File: Find the `Get It Done! To-Do List.html` file.
  • Open in Browser: Double-click the file, or right-click and choose "Open with" your favorite web browser (e.g., Chrome, Firefox, Edge).

3. Key Features and How to Use Them

3.1 Adding a New Task

  1. Type in the Input Field: In the input box at the top, type your task (e.g., "Send email to John", "Prepare presentation slides").
  2. Press Enter: Once you've typed your task, press the **Enter** key. Your task will appear in the list below.

3.2 Marking a Task as Complete/Incomplete

  1. Locate the Checkbox: Each task has a checkbox on its left.
  2. Click the Checkbox:
    • To mark a task as **complete**, click the checkbox. The task text will be struck through and change color.
    • To mark a task as **incomplete**, click the checkbox again. The strikethrough will be removed.

4. Data Storage (Local Storage)

Your tasks are automatically saved in your web browser's **local storage**. This means if you close the browser tab or window and reopen the HTML file, your tasks will still be there.

**Important:** Local storage is specific to your browser and computer. Tasks saved on one computer or in one browser will **not** automatically sync to another.

5. Basic Troubleshooting

  • **Tasks not appearing/saving?** Ensure JavaScript is enabled in your browser. This application relies on JavaScript to function.
  • **Page looks strange?** Make sure the `Get It Done! To Do List CSS.css` file is in the same directory as the HTML file and hasn't been modified.
  • **Buttons not working?** Check that the `Pencil.png` and `Trash.png` image files are in the same directory as the HTML file.

6. Notes for Users

  • **Independent Instances:** Each user will have their own independent "Get It Done! To-Do List." Tasks created by one user will **not** be visible to others.
  • **Browser-Specific Data:** Each user tasks are tied to the specific browser you are using. If you switch browsers or computers, your tasks won't carry over.
  • **Practice Focus:** Feel free to add, edit, and delete various tasks to get comfortable with the application's functionality!
